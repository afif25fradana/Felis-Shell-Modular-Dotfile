# Quick Start Guide

Get up and running with Felis Shell in three easy steps.

### 1. Grab the Files
```bash
git clone https://github.com/afif25fradana/Felis-Shell-Modular-Dotfile.git ~/felis-shell
cd ~/felis-shell
```

### 2. Run the Installer
This will back up your old settings and set up the new ones.
```bash
./install.sh --install-deps
```

### 3. Fresh Start
Restart your terminal or run:
```bash
source ~/.bashrc
```

---

## Pro Tips
- Type shortcut to see everything you can do.
- Type help git or help dev for specific tips.
- Your personal settings go in ~/.bashrc.d/user.conf.
