# Felis Shell

**The terminal setup that works for you, so you don't have to.**

Felis Shell is a collection of modular configurations (dotfiles) designed to make your terminal fast, smart, and beautiful. It automates your environment, provides helpful shortcuts, and looks great doing it.

## Get Started in 60 Seconds

The easiest way to install everything (including dependencies!) on Ubuntu or Arch Linux:

```bash
git clone https://github.com/afif25fradana/Felis-Shell-Modular-Dotfile.git ~/felis-shell
cd ~/felis-shell
./install.sh --install-deps
source ~/.bashrc
```

## Why You'll Love It

- **Smart Prompt** - Shows your Git status, Python/Node versions, and more—only when you need it.
- **Auto-Magic** - Automatically switches Node versions and Python environments when you enter a folder.
- **Modern Tools** - Uses the best modern alternatives like eza (better ls), bat (better cat), and zoxide (better cd).
- **150+ Shortcuts** - From extract for any file to sysclean for a fresh system.
